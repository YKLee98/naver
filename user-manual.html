📋 시스템 개요

Naver-Shopify ERP 시스템은 네이버 스마트스토어와 Shopify 스토어 간의 상품, 재고, 가격을 통합 관리하는 플랫폼입니다.

🔗 실시간 동기화

네이버와 Shopify 간 재고 및 가격 정보를 실시간으로 동기화합니다.

📊 통합 대시보드

양 플랫폼의 판매 현황을 한눈에 확인할 수 있습니다.

🎯 SKU 매핑

서로 다른 플랫폼의 상품을 SKU 기반으로 연결합니다.

💰 자동 환율 적용

실시간 환율을 반영하여 가격을 자동 계산합니다.

🚀 시작하기

시스템 접속

  1. 웹 브라우저를 열고 http://localhost:5173에 접속합니다.
  2. 개발 모드에서는 자동으로 로그인되어 대시보드가 표시됩니다.
  3. 왼쪽 사이드바를 통해 각 메뉴로 이동할 수 있습니다.
💡 팁: Chrome, Edge, Safari 등 최신 브라우저 사용을 권장합니다.

📊 대시보드

시스템의 전체 현황을 한눈에 확인할 수 있는 메인 화면입니다.

주요 기능

  • 통계 카드: 총 상품 수, 재고 현황, 동기화 상태 등 핵심 지표 표시
  • 실시간 활동 로그: 최근 동기화 작업 및 시스템 활동 내역
  • 차트: 재고 추이, 가격 변동, 동기화 성공률 등 시각화
✅ 정상 작동 지표: 동기화 상태가 "연결됨"으로 표시되고 녹색 점이 깜빡입니다.

🔗 SKU 매핑

네이버 스마트스토어와 Shopify 상품을 연결하는 핵심 기능입니다.

새 매핑 추가하기

  1. "매핑 추가" 버튼을 클릭합니다.
  2. SKU 코드를 입력합니다 (예: 2025080501).
  3. Shopify 상품 검색 후 선택합니다.
  4. 네이버 상품 검색 후 선택합니다.
  5. 상품명과 공급업체(vendor) 정보를 확인합니다.
  6. "저장" 버튼을 클릭하여 매핑을 완료합니다.
⚠️ 주의사항: SKU는 양 플랫폼에서 동일해야 하며, 중복된 SKU 매핑은 불가능합니다.

매핑 관리

기능 설명 사용 방법
매핑 수정 기존 매핑 정보 변경 리스트에서 수정 아이콘 클릭
매핑 삭제 불필요한 매핑 제거 리스트에서 삭제 아이콘 클릭
일괄 동기화 선택한 매핑 즉시 동기화 체크박스 선택 후 "동기화" 버튼
필터링 특정 조건으로 매핑 검색 상단 검색창 활용

📦 재고 관리

양 플랫폼의 재고를 실시간으로 관리하고 조정합니다.

재고 조정 방법

자동 동기화

5분마다 자동으로 재고를 동기화합니다. Shopify 재고를 기준으로 네이버 재고가 업데이트됩니다.

수동 조정

특정 상품의 재고를 즉시 조정할 수 있습니다. "재고 조정" 버튼을 클릭하여 수량을 입력합니다.

일괄 업데이트

여러 상품을 선택하여 한 번에 재고를 업데이트할 수 있습니다.

재고 조정 옵션

  • 동기화: Shopify 재고를 네이버에 그대로 반영
  • 절대값 설정: 특정 수량으로 직접 설정
  • 증감 조정: 현재 재고에서 더하거나 뺄 수량 입력
💡 팁: 재고 불일치가 발생하면 "재고 검증" 기능을 사용하여 실제 재고를 확인하세요.

💰 가격 관리

양 플랫폼의 현재 판매 가격을 조회하고 확인하는 기능을 제공합니다.

가격 조회 기능

SKU 매핑 시 가격 확인

SKU 매핑을 설정할 때 네이버와 Shopify의 현재 판매 가격이 자동으로 표시됩니다.

가격 동기화 버튼

"가격 동기화" 버튼을 클릭하면 양 플랫폼의 최신 가격 정보를 불러옵니다.

가격 조회 프로세스

  1. 가격 관리 메뉴로 이동합니다.
  2. "가격 동기화" 버튼을 클릭합니다.
  3. 시스템이 양 플랫폼의 현재 가격을 조회합니다.
  4. 조회된 가격이 테이블에 표시됩니다.
💡 참고: 이 시스템은 가격을 조회만 하며, 자동으로 가격을 변경하지 않습니다. 가격 변경은 각 플랫폼에서 직접 수행해야 합니다.

가격 정보 테이블

항목 설명
SKU 상품 고유 코드
상품명 매핑된 상품 이름
Shopify 가격 Shopify 스토어의 현재 판매 가격 (USD)
네이버 가격 네이버 스마트스토어의 현재 판매 가격 (KRW)
최종 업데이트 가격 정보를 마지막으로 조회한 시간

📈 리포트

판매 현황, 재고 변동, 동기화 이력 등 다양한 분석 리포트를 제공합니다.

리포트 종류

재고 현황 리포트

현재 재고 수준, 품절 임박 상품, 재고 회전율 등

가격 변동 리포트

환율 변동에 따른 가격 변화 추이

동기화 이력

동기화 성공/실패 내역, 오류 상세 정보

매출 분석

플랫폼별 매출 현황, 베스트셀러 상품

리포트 활용법

  • 기간을 선택하여 특정 기간의 데이터만 조회
  • Excel 또는 CSV 형식으로 다운로드 가능
  • 차트와 그래프로 시각화된 데이터 확인
  • 필터를 적용하여 원하는 정보만 추출

🔧 문제 해결

자주 발생하는 문제와 해결 방법

문제: 재고 동기화가 안 됨
  • SKU 매핑이 올바르게 설정되었는지 확인
  • 네이버 API 토큰이 유효한지 확인
  • 상품이 실제로 네이버에 등록되어 있는지 확인
문제: 가격이 자동 업데이트되지 않음
  • 환율 정보가 최신인지 확인
  • 가격 동기화 설정이 활성화되어 있는지 확인
  • 마진율 설정이 올바른지 검토
문제: 상품 검색이 안 됨
  • 검색어가 정확한지 확인 (SKU, 상품명)
  • 네이버에서는 "판매자 상품코드"로 검색
  • Shopify에서는 SKU 또는 상품명으로 검색

오류 코드 설명

오류 코드 의미 해결 방법
401 인증 실패 API 토큰 재설정
403 권한 없음 API 권한 확인
404 상품 없음 SKU 확인, 상품 존재 여부 확인
429 요청 제한 초과 잠시 후 재시도
500 서버 오류 관리자에게 문의
